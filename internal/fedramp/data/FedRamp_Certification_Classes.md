# FedRAMP is departing from the traditional "Low, Moderate, High" labels and transitioning to Certification Classes A through D.

## Where the Baselines Are Now

FedRAMP is in the middle of this structural shift, where requirements and baselines are being streamlined:

- Class A: Initial testing and pilot phases.
- Class B: Replaces the legacy Low / LI-SaaS (156 controls).
- Class C: Replaces the legacy Moderate baseline (323 controls).
- Class D: Replaces the legacy High baseline (410 controls).

For right now, legacy FIPS 199 terms are still widely used while cloud service providers (CSPs) transition to the new programmatic rules and unified "FedRAMP Certified" labels.
